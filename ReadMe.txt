EASIBEX-Python
==============
Windows : 
- Download intervalx_adapt.dll from EASIBEX-MATLAB x86 folder if you use Python 32 bit or x64 folder if you use Python 64 bit and put it in this project folder.
- Run Python in this folder and type : from easibex import *
- Check easibex.py for usage information.

Tested on Windows 8 64 bit using Python 2.7.3 32 bit.
